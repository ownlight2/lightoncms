Own Light CMS

This Netlify-ready CMS accepts a GitHub Developer personal access token for one-time login. After login, edit products, categories, multiple product photos, sizes, colours, discounts, customer purchases, inquiries, Instagram, WhatsApp, fonts, colours, and high-quality CMS-uploaded images.

Create token from GitHub → Settings → Developer settings → Personal access tokens → New token. Paste it once here. The GitHub token is checked by the Netlify Function and is not saved in the browser; only a secure HttpOnly CMS session cookie is saved.

What this CMS controls

Edit fonts, colours, WhatsApp number, Instagram link, homepage hero, category names, products, prices, discounts, size/colour options, stock quantity, product images, multiple gallery images, product video/social/location links, inquiries, visitor tracking, and blog visibility. Saved CMS data is what the public website reads.

SecurityAdmin editing uses a backend HttpOnly session after login. The GitHub token is checked once by the Netlify Function and is not stored in frontend JavaScript.
ImagesUpload high-quality JPG, PNG, GIF, WebP, AVIF, BMP, and safe SVG images up to 25 MB. Uploaded CMS images can be previewed and deleted.
Website linkUse your API URL in the public website config.js. Product changes and image changes show on the website after saving.
